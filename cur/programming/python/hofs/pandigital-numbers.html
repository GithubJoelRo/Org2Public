Exercise 2.1: Pandigital

A number is called pandigital if it uses all of the digits from 1 to n exactly once, where n is the number of digits in the number. So, for example, the five-digit number 15432 is pandigital, but the five-digit number 11132 is not. Again, the four-digit number 4123 is pandigital, but the four-digit number 8312 is not. Finally, the one-digit number 1 is pandigital, but the one-digit number 3 is not. Again for all of the blocks on this page, you should use list comprehension in your solution.

Fill out the function is_pandigital under exercise 2.1 in the starter file. This function should checks if a number satisfies the definition above.

Hint: You should define and implement a factorial function that computes the factorial of a given number

Hint: Using the range function list out all the numbers that should be in a pandgital number of a given length and then check if each of those numbers appears in the number you're checking 


>>> is_pandigital(1)
True
>>> is_pandigital(2)
False
>>> is_pandigital(12)
True
>>> is_pandigital(15432)
True
>>> is_pandigital(11123)
False
