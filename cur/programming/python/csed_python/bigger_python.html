Basic User Input

Let's make a new program. Click on your Sublime text window, and press control-N to create a new file. Then press control-S and save it like we did before. This time name your file "bieber.py".

This time, instead of typing print("Hello World"), you should instead write in the three lines of code below. 

word1 = input("Enter a noun: ")
word2 = input("Enter a past tense verb: ")
print("I heard that " + word1 + " " + word2)

Try running this new program using the terminal. The result should go something like this: 


s273-2 [4] ~/Documents # python3 bieber.py
Enter a noun: Justin
Enter a verb: evaporated
I heard that Justin evaporated.
