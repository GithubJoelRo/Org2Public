Aggregation

Aggregation functions allow us to operate on data in a different way by combining results across multiple rows. Common aggregation functions to be familiar with include COUNT, MAX, MIN, SUM, AVG.

Group Operator

The GROUP operator groups rows that have the same values into summary rows. This brings back the point about how the NATURAL JOIN operator extends the number of columns we have available to us, which can help us identify the patterns in data. By using JOIN on both the Course_Students and Students, there are now multiple rows with the same SID, LEVEL, AGE, FNAME, and LNAME values, since the same student can be enrolled in multiple courses. We can use the GROUP operator on our previous NATURAL JOIN query to make a new query that groups by SID, FNAME, and LNAME, and counts the occurrences of a CID based on the student. This query will allow us to see the student's ID, first name, last name, and number of classes they are enrolled in.
