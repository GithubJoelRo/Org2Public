You should see a 2 appear gloriously in the top right corner. What this code does is it first creates a new empty gameboard of size 4x4, then it places a 2 at position (1,4) in the gameboard (this corresponds to the top right corner), and lastly it updates the display of the screen so that a picture of a 2 actually appears.
