Okay, now that you've learned the basics of object-oriented programming in Python, it is time for you to create your very own class! We will provide you with some basic building blocks and hints, but in the end it's up to you to make it work! Good luck!

If you download the Python file below, you will find a bit of starter code for the class UCBMFET (UC Berkeley Memes for Edgy Teens). Please complete the class so that it has the following characteristics:

To-Do:

  • The following class attributes:
    • members, initialized to 173532
    • topic, initialized to 'memes'
    • posts, intialized to an empty dictionary
  • A constructor method which does the following:
    • sets an instance attribute name to the input paramter of the same name
    • sets two instance attributes activity and posts both to 0.
    • sets an instance attribute member to True
    • increases the class attribute member by 1
  • Completion of the tag_your_friend_in_meme so that it does the following:
    • If the input parameter friend is not an instance of the UCBMFET class, returns string: 'You cannot tag someone in a meme if they are not a member of UCBMFET.' (Hint: You can use isinstance(object, class) to check if an object is an instance of a class)
    • Otherwise it increases your activity (not your friend's) by 1 and returns the string: '@ [insert friend's name here] us in CS10!'
  • Completion of the method post_in_UCBMFET so that it does the following:
    • If the paramter of title_of_post already exists as a key inside the class attribute posts, returns the string, 'You have been banned for reposting a meme.'
    • Otherwise, it adds the title_of_post as a key inside the class attribute posts, with the value initialized to 0 (corresponding to the number of likes so far). It increases your activity and posts each by 1. It also returns the string 'Your total activity on UCMBFET is [insert your activity value here], and your total posts to UCBMFET is now [insert your posts value here].'
  • Completion of the method like_a_post_in_UCBMFET so that it does the following:
    • increases your activity by 1
    • increases the value of the key which corresponds to the post's title in the class attribute posts by 1.
    • returns the string 'Your total activity on UCBMFET is [insert your activity value here, and the total number of likes on the post [insert title_of_post] is [insert number of likes].'

Congratulations! You have just built your very own class!
