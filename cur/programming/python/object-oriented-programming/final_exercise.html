Okay, now that you've learned the basics of object-oriented programming in Python, it is time for you to create your very own class! If you look below your Textbook class, you'll see the class MemePage. Your job is to fill it out!

  • The MemePage class represents one of the many meme pages on the internet. Your job is to fill out the following to-do list so people actually want to post memes on it, and so it has all of the properties of a proper meme page.

To-Do

  1. The following class attributes:
    1. members, initialized to 10000
    2. topic, initialized to 'memes'
    3. posts, intialized to an empty dictionary
  2. A constructor method which does the following:
    1. Sets an instance attribute name to the input parameter of the same name
    2. Sets two instance attributes activity and posts both to 0.
    3. Sets an instance attribute member to True
    4. Increases the class attribute members by 1
  3. Completion of the tag_your_friend_in_meme so that it does the following:
    1. If the input parameter friend is not an instance of the MemePage class, returns string: 'You cannot tag someone in a meme if they are not a member of this page.' (Hint: You can use isinstance(object, class) to check if an object is an instance of a class)
    2. Otherwise it increases your activity (not your friend's) by 1 and returns the string: '@ [insert friend's name here] tagged!'
  4. Completion of the method post_in_page so that it does the following:
    1. If the parameter of title_of_post already exists as a key inside the class attribute posts, returns the string, 'You have been banned for reposting a meme.'
    2. Otherwise, it adds the title_of_post as a key inside the class attribute posts, with the value initialized to 0 (corresponding to the number of likes so far). It increases your activity and posts each by 1. It also returns the string 'Your total activity on this [insert the topic of the meme page here] page is [insert your activity value here], and your total posts to it is now [insert your posts value here].'
  5. Completion of the method like_a_post_in_page so that it does the following:
    1. Increases your activity by 1
    2. Increases the value of the key which corresponds to the post's title in the class attribute posts by 1.
    3. Returns the string 'Your total activity on this [insert the topic of the meme page here] page is [insert your activity value here, and the total number of likes on the post
      [insert title_of_post] is [insert number of likes].'

Congratulations! You have just built your very own class!
