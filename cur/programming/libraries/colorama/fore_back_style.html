Fore

The Fore class has a variety of set attributes which can be used to print text in a different color. For example:print(Fore.RED + 'The text is now red!') will print ‘The text is now red!’ in red. 


    >>> print(Fore.Red + 'The Text is now red!')
    The Text is now red!

Here are all the attributes you will be working with for the Fore class. Have some fun and try printing different text in different colors! 


       Fore: BLACK, GREEN, RED, YELLOW, BLUE, MAGENTA, CYAN, WHITE, RESET

To reset the text color back to default (white), print (Fore.RESET)

Make sure you are always printing when you call these commands because Fore is a class object with attributes, so simply calling it will not produce any side-effect behavior.

Back

The Back class is very similar to the Fore class in the attributes we will be accessing, but rather than changing the color of the text, it changes the background color.

For example: print(Back.GREEN + 'The background is now green!') will print 'The background is now green!’ with a green background. 


    >>> print(Back.GREEN + 'The background is now green!')
    The background is now green!

(HINT: If your text is still red, how could you change the text color back to default?)

The attribute options are the same as those of the Fore class. Have some fun and try printing different text with different background colors!


       Back: BLACK, GREEN, RED, YELLOW, BLUE, MAGENTA, CYAN, WHITE, RESET

Style

The last class we will be focusing on for this lab is Style. Style changes the brightness of your text. For example: print(Style.DIM + 'The text is now dimmed!’) will print 'The text is now dimmed!’ in a lower brightness.

For example: print(Back.GREEN + 'The background is now green!') will print 'The background is now green!’ with a green background. 


  >>> print(Style.DIM + 'The text is now dimmed!')
  The text is now dimmed!

The attributes to change the brightness are listed below. Have fun playing around with it and changing the brightness of your text!

The attribute options are the same as those of the Fore class. Have some fun and try printing different text with different background colors!


Style: DIM, NORMAL, BRIGHT, RESET
